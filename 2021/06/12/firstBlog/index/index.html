新電腦配置

  • 剛好買了一台新電腦，需要重新灌很多軟體，索性這邊紀錄一下，大家也可以參考看看~

VSCode

  • 這是打程式用的文字編輯器，相對於其他編輯器算是平易近人，也有比較多插件支援，所以推薦給想學程式的人~
  • 安裝的時候只要記得依照自己的作業系統安裝就好囉~ 安裝連結
  • 這邊分享我會用到的小插件:
  1. vscode-icons
    它會讓你的 vscode 左側欄位的 icon 更好區別，方便日後在編輯不同類型的檔案時，可以有更直覺式的圖片可以直接點擊。
  2. Prettier
    簡單講它可以讓你的程式碼更能觀看、一致(Readable)。
  3. Live Server
    寫網頁人的特愛，它能讓你在重新編寫網頁時不用一直點開本地端，而是直接在編輯器裡面做及時的網頁更新。
  4. Dark+ Material
    這是 VSCode 的主題樣式，可以上網查查還有更多不同版本，選自己喜歡的就好。
  5. Jupyter
    要定期察看一些輸出值的時候會用到，不過後來它內部有支援cell格式的功能就沒什麼用了，詳情看我，簡單來講只要開頭打上 “#%%” 編輯器裡面就會自動創建一個cell囉~

GitHubDesktop

  • 因為 Github 指令還不熟，菜雞如我還是先下載下來暫時擋擋~哈哈
  • 記得也是要依自己作業系統下載喔! 點我下載

VirtualBox

  • 因為實驗室需求，所以會需要用到 linux 建立網路拓譜那些，想玩其他系統，又不想花大錢直接買實體電腦的，可以下載這個直接玩看看其他作業系統的感覺喔~
  • 這裡偷推一個我喜歡的 youtuber-NetworkChuck 他這部影片有介紹虛擬機，想了解的可以看看~ 有趣影片解釋

Powershell 7、WSL2

  • 此部分只適合 win10 的人喔!
  • 因為 windows 內建終端機我真的是無法忍受…。上網做了功課，發現下載 windows terminal和 Powershell 7 後可以有很好的美化和可讀性，甚至跑子系統 linux 在裡面，所以一不做二不休，看了此篇教學就一併裝進這台電腦。

Obsidian

  • 一個很好用的卡片式筆記軟體，因為是安裝在主機裡，所以可以離線使用。它還會自動幫你生成關聯圖，在複習的時候會非常有效率，個人目前論文參考文獻都會用這個軟體來寫。
  • 介紹下載點

Notion + Notion export tool

  • 一款很好用的百寶工具，它可以規畫行程、專案管理等等，而且也提供學生帳號無上限大小上傳的服務，非常的好用，真心大推!
  • 有時候也會需要將自己的行程或是筆記印出來，網路上有找到一個免費的輸出資源，這裡下載，教學點我，這樣就不用買會員也能輕鬆輸出啦~

Extra

  • Anki、PacketTracer: 因為要想考取 CCNA 200-301 證照，所以下載這些程式來練習，網路上也有很多佛心的人願意免費教學，大推這一位 youtuber 的教學!
