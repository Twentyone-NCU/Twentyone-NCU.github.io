Notion 介紹、日常使用分享

發表於2021-12-19|更新於2021-12-19|讀書筆記分享
|閱讀量:

Outline

  • 在這裡會為大家介紹我平常使用的工具 – Notion，會介紹自己常用的功能給大家，幫助大家可以快速入手，介紹順序分別從:
  1. 常用基本物件添加
  2. Markdown 語法、快速作筆記法、跟一些覺得好用的快捷鍵
  3. Notion 獨特 Database view 的功能
  4. 自己生活搭配使用方式，
  5. 總結

基本添加物建介紹

基本入門介紹

  • 首先，為了方便大家快速了解，直接上影片!其實PAPAYA - Youtube就有很不錯的介紹了，大家可以先看完，再繼續看下去以下介紹~!

常見物件介紹

  • 看完影片後重新幫各位整理，以及分享我的使用方式，首先，先上個我個人首頁圖~

  • 如同前面 PAPAYA 老師介紹的，最基本 Header、Bullet point、toggle、page 大家使用上都很直接我這邊就不多做贅述。比較想提及的部分有 quote、call out、image input、table、equation、code 這些功能，以下分別一一介紹:

    1. quote: quote 功能我通常除了拿來放名人佳句，也會放一個筆記的開頭 (Header 1下方)。算是當作一個段落的筆記的重新整理，最精華重點。濃縮成一句話，以便之後複習可以快速想起。
    2. call out: call out 有很顯得文字框搭配 icon，通常我會拿來放靈光一閃的想法或疑問，以便之後複習的時候可以特別看到，再進一步去做動作。
    3. image input: 很直白的功能，就是能導入圖片，除了能複製貼上、上傳圖片之外，它也支援 link 連結，也就是說你只要有那個徒聯的連結，它也能直接讓它顯現出來。
    4. table: 這是 Notion 最近新加的功能，它不是 data base 的 table，就是大家能直覺上想到最簡單的那種基本 table，目前還不能做太多客製化功能，但我相信它之後會慢慢增加它的多工性的。
    5. equation: 這有無數的用法，其中兩項最常用的就是拿來寫數學算是，它能讓一些數學符號直接顯現出來(markdown本身語法)。另外一項就是拿來做自己客製化! 就像首頁那樣的標題字深紅色背景，那就是叫出 equation 裡面的 colorbox 功能，大家有機會也能試試看~
    6. code: 這功能我目前都用來做程式碼的紀錄，它有支援很多功能，也都有高光字體，很適合和同事做協同合作使用，當然自己幫自己複習也很有幫助~!

  • 最後在這邊做小小額外分享，我個人首頁的 cover 跟下面圖片的部分都是用 Canva 去做，大家若想客製化自己的首頁，十分推薦這套軟體，它也有網頁版，重點是學生帳號免費!!! 還不好好把握，趕快用😆

Markdown 語法, 快速作筆記, 快捷鍵

Markdown Cheatsheet 參考

  • Markdown 雖然口頭說語法，但其實我不認為它是一種語法，就只是一種做筆記的方式而已，所以不是寫程式的人也千萬不要怕，加上 Notion 本身有多加很多友善功能，讓你能在不知不覺中不用 Markdown 也能用到 Markdown 功能喔!首先，先分享我常用的參考網站:
    中文版 Markdown 語法 數學符號 Markdown
    Markdown 物件對應按法 Markdown 數學符號按法
  • 有很多筆記軟體也都是用markdown，會一個notion 之後就很容易上手其他的，像是 obsidian 這些，是真的十分推薦一定要學一下!!!!

Notion 小體貼

  • 如果真的不會的，也可以打 ”/” 叫出想要的物件，甚至是靠形狀打出來! 按出形狀再按空白鍵之後就會顯示出來!以下為例子:
    • “|” → 叫出 quote
    • “[” + “]” → 叫出 to do
    • “-” + “>” OR “=” + “>” → 叫出箭頭符號
    • “*” → 叫出 bullet point
    • “#” → 叫出 header
    • 三個 “-” → 叫出分隔線

額外好用快捷鍵分享

  • Notion 也有很多其他好用快捷鍵，這邊分享幾個我最常使用的!
    • Ctrl + Shift + L → Dark / Light mode 切換
    • Ctrl + P → 叫出想要的 page (類似 vscode 的搜尋欄)
    • Ctrl + B / U / I → 粗體字 / 底線 / 斜字體
    • Ctrl + H → 套用剛剛顏色字體
    • Ctrl + M → 添加評論功能
    • “:” + “符號名稱” → 叫出 emoji

Data base view

個人覺得 Notion 最屌功能

最厲害的就是同個資料庫，它是可以換視角方式去看的!!

  • Notion Data base 有分為 table、board、gallery、list、calendar、timeline，以自己 book shell 為例，不僅可以視覺化我的圖書資料庫，也可以加 tag 去做不同的書本類型篩選(左上角 view 模式)。以下為我同一個書庫，加上不同 tag 或是 base type 模式去檢視自己的書庫!

    List 模式 Gallery 模式 Gallery+tag 分類
    可以比較快速瀏覽自己書籍種類並確定有沒有看過 可以以圖片方式做瀏覽動作 加上TAG分類後更能一目了然想找的書籍

  • 以上只是一個例子，你都可以按照自己的習慣去做自己的資料庫，真的會一目了然，對自己的專案管理會更輕鬆! 我已經不只圖書資料庫，包括習慣紀錄、跳舞套路表、收支表、自己的專案進度追蹤等等，都用這種方式了，所以有打算用 Notion 的各位，務必學好這個!雖然可能有學習曲線，但我可以保證，學熟了後，絕對不會後悔!!!

生活使用方式

  • 講完以上東西，我就來分享我是怎麼在日常生活中，搭配 Notion 進行我的日常生活管理吧~!

    生活管控方面

    工具: Agenda、Apple calendar、Calendars app、Apple Reminder

  • 在管理自己的時間方式前，我想先提，有各式不同的方法，但就像 Atom Habit 裡面所說的，不要找最受歡迎的那個，找最適合自己的那個!所以我先來講我自己的管理方式，再進而去做管理執行的動作。

    需求: 我很懶、記憶力很差，但又不想很死的在”確切時間”做”指定的事情”

    1. Time block 模式: 我會將事情分 priority 等級顏色劃分，然後大致定上一個時間區段，讓我有心理準備今天大概要做哪些事情，集中多少精神去完成今天的事。
    2. 保持彈性: 意外很容易突然發生，那這時候我都會有保持彈性，每當有當下要額外的事情時，我絕對當下就直接語音叫出 Siri 幫我記錄在 Reminder 裡面，等到我比較有空閒的時候，再進入 Notion 今天的 Agenda 裡面做更改。
    3. 保持提醒: Agenda 做完後，我還是會再把它放入 Apple Calendar 裡面，搭配 Calendars app 的 widget (免費!)，可以做隨時提醒的動作。畢竟我這人喜歡當下就做一件事，實在不想無時無刻都在想著一整天的行程😂😂
    4. 週規劃重點: 我會將一年目標切分成月目標，再切分成週目標，再切分成日目標。每週日晚上，都會在 Weekly agenda 裡面做對應的行程規劃執行，這樣除了能有效執行目標外，也能隨時確定做的事情有沒有偏離軌道!

總結

自己使用後改變

  • 我大概是在碩一上接觸到 Notion 慢慢自己摸出來，一直到碩一下開始有很明顯的改變，以前總是東做西做，忙東忙西，但能量很分散，這也導致很多事情都半吊子。自從有了 Notion 後，真的是世界都不一樣。我本性一樣懶跟健忘，但有這些東西的幫忙，真的幫我省去很多需要花費精神去做的事，所以才能集中能量在真正重要的事情上，科技可以讓我們各種分心，但同時它也可以讓我們有更好的生活進步啊!!

    懷海德: “讓可以不假思索執行的事情變多，就是文明進步的基礎”
