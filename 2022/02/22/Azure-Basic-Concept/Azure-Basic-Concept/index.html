Azure Basic Concept

發表於2022-02-22|更新於2022-02-22|雲端知識
|閱讀量:

What to Expect?

  • In this section we will know about Azure basic concept.
  • We’re going to dive into each categories, which are:

    1. Adoption Journey

    2. Manage components

    3. Interacting tools

    4. Global Architectural Components

    5. Service types

Adoption Journey

  • Adoption Journey is about the steps about how to implement Azure with your situation.

Manage components

  • Manage components is about how & why to manage your account & different resources with your account.
  • We can learn about the hierarchy structure and several different managing ways.

Interaction tools

  • Interaction tools is about how you can control through different kind of tools to control your Azure cloud resource.

Global Architectural components

  • Global Architectural components is the physical structure in Azure
  • The goal of this is to achieve the “High availability” purpose

Service types

  • There are many different service types in Azure and we’re going to overview most of them with a representative service! Even more, you can do you’re own exercise!
