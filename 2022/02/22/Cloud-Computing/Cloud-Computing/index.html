Why Cloud Computing?

  • There are a lot of cloud provider in current time, like AWS, Azure, and GCP are three main cloud providers right now.
  • There are many advantage about using cloud computing and you can break it into three main abstract:

    Expenditure types advantage

    Deployment Model advantage

    Service model advantage

How to dive into and learn quickly in each cloud provider

  • Here we provide a visualize roadmap to help you learn the resource!
  • You can click on the node that you are interest in and flow the path, soon you will be a cloud expertise!
  • Also, there are many basic certification to test your basic knowledge in different cloud provider like
  1. Azure: Azure, AZ900
  2. Amazon: AWS, Certified Cloud Practitioner/CLF-C01
  3. Google: GCP, Associate Cloud Engineer

Framework to learn faster

  • In the later content, you can keep this framework in mind to help you learn each cloud provider concept faster!
  1. Infrastructure of the components
  2. Interacting tools with it own cloud resources
  3. Adoption Journey of implementing the cloud resources
  4. Service Categories of itself services
  5. Manage component about billing, security, … etc.
